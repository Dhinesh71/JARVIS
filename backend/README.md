# JARVIS Backend

Node.js + Express backend for the JARVIS conversational AI.

## Setup

1.  Navigate to this directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file:
    ```bash
    cp .env.example .env
    ```
4.  Add your Groq API Key to `.env`.

## Run

```bash
npm start
```

Server defaults to `http://localhost:5000`.
